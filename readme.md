# Template sass

Sass is the most mature, stable, and powerful professional grade CSS extension language in the world.

## Installation

Install node.js https://nodejs.org/en/ then install dependencies node


```bash
npm i
```

## Usage

```
npm run sass
```

```
npm run sass-build
```
